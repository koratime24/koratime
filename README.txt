📁 الملفات الموجودة في هذا المجلد:

  📄 index.html              → الصفحة الرئيسية (كل زائر جديد يراها أولاً)
  📄 about.html              → صفحة "من نحن"
  📄 contact.html            → صفحة "اتصل بنا"
  📄 privacy.html            → صفحة "سياسة الخصوصية"
  📄 disclosure.html         → صفحة "إفصاح العمولات"
  📄 article-template.html   → قالب لكتابة المقالات الجديدة
  📄 style.css               → ملف التنسيق المشترك لكل الصفحات
  📄 README.txt              → هذا الملف


═══════════════════════════════════════════════════════════
  🚀 كيف ترفع الموقع إلى Netlify؟
═══════════════════════════════════════════════════════════

  1. افتح جميع ملفات الـ HTML + style.css في مجلد واحد.
  
  2. اذهب إلى netlify.com وسجل دخول.
  
  3. في لوحة التحكم، اختر مشروع koratime24.
  
  4. اضغط على "Deploys" → "Drag and drop your site output folder here".
  
  5. اسحب مجلد koratime_site كاملاً إلى المربع.
  
  6. انتظر 10-30 ثانية حتى يتم الرفع.
  
  7. افتح koratime24.org للتأكد من أن كل شيء يعمل.


═══════════════════════════════════════════════════════════
  ✍️ كيف تكتب مقالاً جديداً؟
═══════════════════════════════════════════════════════════

  1. انسخ ملف article-template.html وأعطه اسماً بالإنجليزية:
     مثلاً: real-madrid-vs-barcelona-april-2026.html
     
     ⚠️ مهم: استخدم شرطات - بدل المسافات، وابقَ بالإنجليزية
     هذا يساعد في تحسين ترتيب المقال في جوجل.
  
  2. افتح الملف الجديد في أي محرر نصوص (VS Code، Notepad، TextEdit).
  
  3. استبدل كل النصوص بين علامات [ ] بمحتوى مقالك الحقيقي:
     - عنوان المقال
     - الـ meta description (مهم جداً لجوجل)
     - محتوى المقال
     - الوسوم (tags)
  
  4. احفظ الملف في نفس مجلد باقي الملفات.
  
  5. ارفع المجلد كاملاً مرة أخرى إلى Netlify (خطوة 4 و5 أعلاه).
  
  6. مقالك الآن متاح على: koratime24.org/اسم-الملف.html
